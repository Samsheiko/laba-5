# # Шейко Семён Группа ИТС-1 Лабораторная №5







## Задание 1







### Задача 1







### Текст задачи








#### Составить программу, которая находит значение отношения трехзначного числа к сумме его цифр.







### Алгоритм решения



######



#### Алгоритм работы программы начинается с вызова предиката start, который инициирует ввод данных через стандартный поток read. Полученное значение X проходит проверку на соответствие типу данных (integer) и заданному диапазону трехзначных чисел (от 100 до 999). Далее число разбивается с помощью целочисленного деления и остатка от деления. Полученные цифры суммируются в Sum. После этого находим отношение исходного числа к сумме цифр. В завершение программа выводит значения суммы и отношение.

# Задание 2







###



### Текст задачи







#### Определить количество ненулевых элементов списка.







### Алгоритм решения







#### Алгоритм работы программы начинается с вызова предиката start. Далее основной предикат разделяет список на голову (первый элемент) и хвост (остаток). Если список пуст, возвращается ноль. Для непустого списка программа рекурсивно вычисляет количество ненулевых элементов в хвосте, после чего проверяет голову: если она не равна нулю, к общему счетчику прибавляется единица, иначе значение остается прежним. Итоговый результат выводится на экран.
