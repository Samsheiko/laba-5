### Задача 1







### Текст задачи







#### Разработать класс с тремя вещественными полями. Создать конструктор копирования.
Разработать метод, приводящий поля к целому типу. Перегрузить метод ToString() для
формирования строки из полей класса. Реализовать дочерний класс (его содержание предложить
самостоятельно и описать в решении: какой содержательный смысл имеют поля; реализовать
конструкторы; предложить и реализовать 2-3 метода). Протестировать все конструкторы и
другие методы базового и дочернего классов.








### Алгоритм решения



######

##### Алгоритм начинается с ввода пользователем марки автомобиля и числовых параметров расхода. Считанные значения преобразуются в тип double для выполнения расчетов. На основе этих данных инициализируется объект класса Car. Далее вызываются вычислительные методы: первый рассчитывает максимальную дистанцию по трассе, второй — средний расход топлива. Программа также демонстрирует работу конструктора копирования, создавая дубликат параметров, и метода ToInteger, который приводит дробные значения к целому типу. В завершение результаты вычислений и состояние объекта выводятся на экран.

# Задание 2







###



### Текст задачи







#### Определить количество ненулевых элементов списка.







### Алгоритм решения







#### Алгоритм работы программы начинается с вызова предиката start. Далее основной предикат разделяет список на голову (первый элемент) и хвост (остаток). Если список пуст, возвращается ноль. Для непустого списка программа рекурсивно вычисляет количество ненулевых элементов в хвосте, после чего проверяет голову: если она не равна нулю, к общему счетчику прибавляется единица, иначе значение остается прежним. Итоговый результат выводится на экран.
