# # Шейко Семён Группа ИТС-1 Лабораторная №5







## Задание 1







### Задача 1







### Текст задачи







#### Составить программу, которая находит значение отношения трехзначного
числа к сумме его цифр.







### Алгоритм решения



######

##### 1. Пользователь вызывает: предикат start. для запуска программы. 2. Подсчёт количества цифр в числе X: выполняется через вызов sum_digits(X, Sum), где трехзначное число математически разбивается на сотни (// 100), десятки (// 10 mod 10) и единицы (mod 10) для получения их суммы.  3. Вычисление отношения параметров: выполняется через деление исходного целого числа X на вычисленную сумму его цифр (Otn is X / Sum).  4. Проверка диапазона и вывод результата: программа проверяет, является ли ввод трехзначным числом от 100 до 999, и выводит результаты Sum и Otn, либо сообщает об ошибке и рекурсивно перезапускает ввод.

# Задание 2







###



### Текст задачи







#### Определить количество ненулевых элементов списка.







### Алгоритм решения







#### Алгоритм работы программы начинается с вызова предиката start. Далее основной предикат разделяет список на голову (первый элемент) и хвост (остаток). Если список пуст, возвращается ноль. Для непустого списка программа рекурсивно вычисляет количество ненулевых элементов в хвосте, после чего проверяет голову: если она не равна нулю, к общему счетчику прибавляется единица, иначе значение остается прежним. Итоговый результат выводится на экран.
